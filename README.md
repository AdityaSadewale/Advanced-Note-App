````

```bash

```

3. **Open `index.html` in your browser**

## 💻 Usage

* Enter the default password: `1234` to unlock the app.
* Click **+ New** to create a note.
* Use the **🎨** button to change note colors.
* Pin important notes with **📌**.
* Delete notes with **✖**.
* Use the search bar to find notes instantly.
* Toggle dark mode with the **🌙** button.
